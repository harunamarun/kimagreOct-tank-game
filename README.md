## Setup Instructions
#### Clone the repo:
```
git clone https://github.com/harunamarun/tank-game.git
```

#### Install packages:
```
yarn
```

#### Start server:
```
yarn start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
